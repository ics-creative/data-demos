# スクリプト処理の分散
***
<br>
##2. <span class="red">レンダリングをマニュアルで行い、画面描画と計算処理を分離する</span>

<br>
### 演算処理、ロジック処理などの実行時間が長いフレームでは画面描画はせずに、オブジェクト等の計算処理のみ行う。

```actionscript

// マニュアルレンダリングに切り替え
starling.shareContext = true;

// レンダリングする。 (ループ関数内)
if( isRenderable ){
    starling.context.clear(bgColor, 1);
    starling.nextFrame();
    starling.context.present();
}
```
