GameInput Character Demo (Flash Stage3D)

July. 2013

How to Play

Connect Gamepad

  • First Stick : Move Character
  • Second Stick(if exsits) : Move Camera
  • Any Buttons : Jump

Spec

  • This is Flash Stage3D contents.
  • Using Away3D 4.1.0 Framework
  • Using GameInput API that needs Flash Player 11.8.0 or over
  • Chrome PPAPI may not find gamepad
