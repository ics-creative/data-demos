Gamepad Use Demo (Flash Stage3D)

December. 2014

How to Play

Install the Adobe AIR runtime application on the Android device.

Both the Android device and the desktop computer need to be on the same internet network.

  • Tilt Device : Move Character
  • Touch Device's screen : Move Camera
  • Shake Device : Jump

Spec

  • This is Flash Stage3D contents.
  • Using Away3D 4.1.0 Framework
  • Using AIRGamepad API that needs Flash Player 11.3.0 or over
